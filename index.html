Introducing

Pitt Fuego

A Multimodal Communication Tool

A simple solution for promoting coding literacy

Multimodal work in the composition classroom offers the perfect opportunity for you and your students to start working with code. "Pitt Fuego" will help you get started…and using this tool is easy, even if you’ve never worked with code before. With Pitt Fuego, you and your students will use code to generate attractive multimodal websites following design best practices.









You can do this!

But why work with code in the composition classroom?...well, because code is already everywhere, writing its way into in everything we do—it’s even generating the words you’re reading right now. While written code might look scarry, or seem too difficult to learn, teach, or apply, it needn’t be! With Pitt Fuego, all of the instructions for learning to work with the code are embedded in the code itself—so there’s nothing for you to do in terms of teaching other than learn the basics of how to use the tool and encourage students to work together to solve problems.

What's the data say?

Our early data from this project show that working with code is easier than you think, and that using this tool in the classroom will improve your students’ coding literacy, computational thinking, and understanding of computer science. Ultimately, working with code in the composition classroom will lead to new possibilities for composing and new pathways for student empowerment.





Easy Integration

Have no fear! You’ll still be doing the things you normally do in composition classes and multimodal work, like inventing, arranging, styling, thinking of better ways to connect with audiences and circulate media content. But now, instead of relying on third parties like Wix and Weebly, your students will modify and control their own code. If Pitt Fuego isn’t easy to learn and use, then something needs to be fixed on our end…we welcome your feedback!
